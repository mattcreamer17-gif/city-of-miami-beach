# City of Miami Beach — Landing page

Intended URL path: /for/city-of-miami-beach

A single-page briefing prepared for the eMerge Americas 2026 booth conversation with the City of Miami Beach. Built in the city's own visual vocabulary (navy masthead, Futura-PT/Montserrat type, near-square corners, yellow used sparingly) so the page reads like something the city's own marketing team would ship. Content is drawn from the FY2026 IT department posture, the Resilient Florida grant register, and the city's public dashboards. The primary CTA routes to Matt Creamer's 30-minute Calendly.

Files: `index.html`, `styles.css`, `logo.svg`.
